# Este é o meu gerador de códigos ean nacional
Este repositório contém um gerador de códigos EAN (European Article Number) nacional, que pode ser utilizado para criar códigos de barras únicos para produtos comercializados no Brasil.
## O que é um código EAN?
O código EAN é um sistema de numeração utilizado para identificar produtos de forma única em todo o mundo. Ele é composto por 13 dígitos, sendo os primeiros três dígitos o prefixo do país (no caso do Brasil, é 789), os próximos quatro dígitos representam o fabricante, os próximos cinco dígitos identificam o produto e o último dígito é um dígito verificador, calculado a partir dos outros 12 dígitos.
